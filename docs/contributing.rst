Contributing
============

You can contribute by filing issues and by submitting pull requests.

Suggested improvements:
- Convert the application into a small package with a proper module name (no spaces in filename).
- Add unit tests for parsing and scanning logic.
- Improve unit/functional tests for ScanThread.
- Add more UDP/TCP probes and better heuristics for banner parsing.

Please follow standard GitHub flow:

1. Fork repo
2. Create feature branch
3. Submit PR to `main` with description and testing details.

Thank you.
